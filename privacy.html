🛡️ BizBunker 개인정보처리방침

100% OFFLINE · ZERO DATA COLLECTION

본 문서는 BizBunker(이하 "앱") 이용자의 개인정보 보호에 관한 사항을 안내합니다.

1. 수집하는 개인정보 항목

본 앱은 사용자의 개인정보를 일절 수집하지 않습니다.

  • 앱 내 모든 데이터(재무 정보, 거래처 정보, 설정값 등)는 사용자의 기기 내부(로컬 SQLite 데이터베이스)에만 저장됩니다.
  • 외부 서버로 전송되는 데이터는 없습니다.
  • 분석 도구(Analytics), 광고 SDK, 추적 코드가 일절 포함되어 있지 않습니다.

2. 네트워크 통신

본 앱은 다음의 경우에만 선택적으로 네트워크를 사용합니다:

  • 환율 정보 조회: 사용자가 기능을 실행할 경우 공개 API에 요청을 보내며, 개인정보는 전송되지 않습니다.
  • 인앱 구매: 앱 구매를 위해 Apple의 StoreKit 시스템과 통신합니다.

상기 기능을 제외하고 앱은 완전히 오프라인으로 작동합니다.

🛡️ BizBunker Privacy Policy

100% OFFLINE · ZERO DATA COLLECTION

This document outlines the privacy practices for the BizBunker application.

1. Data Collection

This app does NOT collect any personal user data.

  • All app data is strictly stored locally on your device within a secure SQLite database.
  • No data is transmitted to external servers.
  • The app contains zero analytics tools, advertisement SDKs, or tracking codes.

2. Network Usage

The app only uses the network selectively for:

  • Exchange Rates: Fetching public exchange rates (no user data is sent).
  • In-App Purchases: Communicating with Apple's StoreKit for premium features.
